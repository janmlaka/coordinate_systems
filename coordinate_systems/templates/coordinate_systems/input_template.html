READ ME

For the website to work properly you have to set in the following files:

-D48/GK file

-D96/TM file

The file with D48/GK coordinates has to meet the next conditions:

The coordinates have to be in following order: X Y h

The standart deviations also have to be in the following order: X Y h

The file with D96/TM coordinates has to meet the next conditions:

The coordinates have to be in following order: e n h

The standart deviations also have to be in the following order: e n h

The heights have to be elipsiod heights.

Once you setup the files you may click "Calculate"
